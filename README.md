# Live-Website-IP-Checker
Harici DNS sorgusu yapmadan, tarayıcı API'si ile bağlandığınız gerçek sunucu IP'sini anlık gösteren hafif bir araç. DNS yayılımı ve CDN testleri için ideal.  / Lightweight tool showing real server IP via browser API, avoiding external DNS lookups. Perfect for DNS propagation and CDN testing.
